🚀 Supabase Database Setup for BookFlow

⚠️ Important: This setup will open Supabase in a new tab. Follow the steps below to get your database connection string.

Step 1: Create Supabase Account

Click the button below to open Supabase and sign up for a free account:

Step 2: Create New Project

  1. Click "New Project" in the dashboard
  2. Choose your organization
  3. Enter project name: bookflow-db
  4. Set a strong database password
  5. Choose a region closest to you
  6. Click "Create new project"

⏱️ This takes 2-3 minutes to provision

Step 3: Get Connection String

Once your project is ready:

  1. Go to Settings → Database
  2. Scroll down to Connection string
  3. Copy the Pool connection string
  4. It looks like: postgresql://postgres:[YOUR-PASSWORD]@db.******.supabase.co:5432/postgres

Step 4: Update Your Environment

Edit server/.env and replace the DATABASE_URL:

DATABASE_URL=postgresql://postgres:[YOUR-PASSWORD]@db.******.supabase.co:5432/postgres

Step 5: Apply Database Schema

  1. In Supabase dashboard, go to SQL Editor
  2. Open the file server/db/schema.sql in your code editor
  3. Copy the entire SQL content
  4. Paste it in the Supabase SQL Editor
  5. Click Run to execute

✅ All Set!

Now you can start your development servers:

npm run dev

Your app will be available at:

  • 🌐 Frontend: http://localhost:3000
  • 🔧 Backend API: http://localhost:5000/health
💡 Pro Tip: Save your Supabase project URL and password securely. You'll need them if you want to deploy your application later.
